# AIER Monetary Rules Dashboard

This public repository hosts the static AIER Monetary Rules Dashboard.

[Open the live dashboard](https://matthewschafferecon.github.io/aier-monetary-rules-dashboard/)

The site provides interactive monetary-policy rule estimates, historical policy signals, methodology and source notes, and a downloadable XLSX dataset. The repository contains only the generated files needed to operate the public dashboard.

## Published files

- `index.html`: dashboard application
- `data/results.json`: current and historical dashboard data
- `data/monetary-rules-dashboard-data.xlsx`: downloadable workbook
- `data/archive/`: immutable public release snapshots and their manifest

Updates are published only after the source project passes its numerical, archive-integrity, workbook, and browser validation checks.
